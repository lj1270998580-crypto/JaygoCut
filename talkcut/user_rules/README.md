## AI 审核顺序（按优先级）

1. 静音 `>=1s`：建议删除（整段标记）
2. 残句：删除（话说一半 + 停顿）
3. 重复句：删除较短版本（优先保留后一句完整表达）
4. 句内重复：删除重复片段（A+中间+A）
5. 卡顿词：按规则删除
6. 重说纠正：删除前面错误表达
7. 语气词：标记待人工确认

## 核心原则

删前保后：后说的通常更完整，优先保留后半段表达。
