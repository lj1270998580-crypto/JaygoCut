# 静音段处理

## 阈值规则

| 静音长度 | 处理 |
|---------|------|
| < 0.2秒 | 忽略（自然停顿） |
| >= 0.2秒 且 < 1秒 | 可选删除（句间停顿） |
| >= 1秒 | 建议删除（明显卡顿或展示画面） |

## 输出格式

整段标记，不拆分。

示例：3.2 秒静音 -> 输出 1 条。

```md
| 64-66 | 12.86-15.80 | 静音3.2s | | 删 |
```

用户可在审核页面手动取消不想删除的静音段。

## 特殊情况

### 长静音
连续 5s+ 的静音整段标记，预选删除。

### 开头静音
视频开头静音默认删除。
