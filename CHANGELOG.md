# Jaygo Cut Changelog

## 0.1.14

- 审核页新增“预听当前删除点”按钮，支持 `S` 快捷键，可播放删除片段前后约 2 秒，并高亮当前预听片段。
- 新增“删除诊断”面板，统计删除片段总数、删除总时长、最长/最短片段、连续密集删除段和风险片段。
- 风险片段支持点击跳转，便于快速检查过短、过长、紧贴保留词等容易残音或吞字的位置。
- 执行裁剪前强制保存当前审核状态，并生成 `review-state.backup.json`，降低裁剪失败或异常关闭导致丢失数据的风险。
- 新增剪辑精度模式：`保守`、`标准`、`干净`。默认仍为 `标准`，只在提交裁剪任务时调整删除边界，不改变用户看到的文本时间戳。
- 裁剪失败弹窗改为更易理解的原因提示，详细错误保留在日志中，并增加“复制诊断信息”能力。
- 历史记录增强：检测原视频是否缺失，支持重新定位视频；视频缺失时仍可打开只读审核页查看文本和状态。
- 扩充回归测试，覆盖删除诊断、剪辑模式边界、裁剪前备份、历史记录健康状态和审核页新增入口。

## 0.1.13

- 主界面设置移除“图片比例”，避免和审核页的视频配图比例重复；图片比例现在统一在审核页选择。
- 保留审核页图片比例传参，生成图片时仍按审核页当前选择生效。
- 包含此前审核页快捷键指南、文本搜索替换、剪映草稿导出、LLM/视频配图等修复。

## 0.1.12

- 审核页新增快捷键指南按钮，支持 `Space` 播放/暂停、`Ctrl+Z` 多步撤回、`Ctrl+Y`/`Shift+Ctrl+Z` 重做、`Ctrl+F` 搜索纠错、`Ctrl+S` 保存草稿。
- 审核页新增文本搜索与批量替换，可用于纠正“他/她/它”和专有名词识别错误。
- 补全剪映草稿导出，生成草稿目录，并支持内置字幕样式和用户自定义字幕模板。
- 修复审核页左右悬浮按钮在非全屏或窄窗口下重叠的问题。
- 优化 LLM 口播剪辑提示词，降低误删有效内容的概率。
- 在线更新模块新增“本次更新内容”展示，支持从服务器 `release-notes.json` 读取说明。

## 0.1.10 - 0.1.11

- 修复审核页空白、历史审核恢复、波形缩放和播放光标跳动问题。
- 优化静音阈值、撤销、拖选和边界补偿，减少剪辑后残留音。
- 增加 SRT/TXT 字幕导出和视频配图实验功能。

## 0.1.8 - 0.1.9

- 将运行时输出目录改为英文，降低 Windows 中文路径编码风险。
- 增加阿里 Qwen3-ASR 异步识别和本地模型检测/安装流程。
- 增加在线更新基础能力。
