# Jaygo Cut Changelog

## 0.1.13

- 主界面设置移除“图片比例”，避免和审核页的视频配图比例重复；图片比例现在统一在审核页选择。
- 保留审核页视频配图比例参数传递，生成图片时仍按审核页当前选择生效。
- 补充回归测试，确保主界面不再暴露图片比例设置。

## 0.1.12

- 审核页新增快捷键指南按钮，支持 Space 播放、Ctrl+Z 多步撤回、Ctrl+Y/Shift+Ctrl+Z 重做、Ctrl+F 搜索纠错、Ctrl+S 保存草稿。
- 审核页新增文本搜索与批量替换，可手动修正“他/她/它”和专有名词，修正内容会进入草稿、SRT/TXT 和剪映草稿导出。

- 补全剪映草稿导出：生成草稿目录，支持内置字幕样式和用户自定义字幕模板草稿。

- 修复审核页左右悬浮按钮在非全屏或窄窗口下重叠的问题。
- 优化 LLM 口播剪辑提示词：先理解主题和结构，保护事实、数字、因果、转折、结论，只标记高确定性冗余内容。
- 提高 LLM 删除建议置信门槛，减少把有效内容误标为删除的情况。
- 在线更新模块新增“本次更新内容”展示，支持从服务器 `release-notes.json` 读取更新说明。
- 同步 README、软件介绍和交接文档。

## 0.1.10 - 0.1.11

- 修复审核页空白、历史审核恢复、波形缩放和播放光标跳动问题。
- 优化静音阈值、撤销、拖选和边界补偿，减少剪辑后残留音。
- 增加 SRT/TXT 字幕导出和视频配图实验功能。

## 0.1.8 - 0.1.9

- 将运行时输出目录改为英文，降低 Windows 中文路径编码风险。
- 增加阿里 Qwen3-ASR 异步识别和本地模型检测/安装流程。
- 增加在线更新基础能力。
