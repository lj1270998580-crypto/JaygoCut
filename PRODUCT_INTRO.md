# Jaygo Cut 软件介绍

## 软件定位

Jaygo Cut 是一款面向中文口播视频创作者的桌面端剪辑工具，重点解决长口播素材“听一遍、找废话、手动切点、反复导出”的低效问题。

核心流程：选择视频 -> 自动转录 -> 审核页确认 -> 精确裁剪 -> 导出成片和字幕。

## 核心能力

- 云端转录：火山引擎 ASR、阿里 DashScope Qwen3-ASR 异步识别。
- 本地转录：支持本地 Whisper/ggml 模型，不把大模型内置进安装包，用户可一键检测和安装。
- 规则预标记：静音、语气词、重复句、卡壳和明显重说。
- LLM 标记：先理解主题和文章结构，再保守标记可删内容，避免误删主线。
- 审核页体验：音频播放、波形、逐字文本、颜色标记、撤销、拖选、状态保存。
- 发布辅助：生成标题、简介、关键词；通过对话调整 LLM 标记。
- 视频配图：根据内容生成配图点、分镜画面提示词和图片预览。
- 导出能力：成片、SRT 字幕、TXT 文案、剪映草稿目录；剪映草稿支持填写自己的字幕模板草稿目录来复用样式。
- 在线更新：应用内检查、下载并安装更新，可显示本次版本更新内容。

## 使用流程

1. 在设置页选择转录引擎，填写火山/阿里 API Key 或检查本地模型。
2. 选择视频和输出目录，点击“开始完整流程”。
3. 打开审核页，检查自动标记、调整静音阈值、按需运行 LLM 标记。
4. 人工确认后执行裁剪。
5. 在历史记录中打开项目、恢复审核或删除记录。

## 适合人群

- 做口播、知识分享、观点类短视频的创作者。
- 需要批量处理课程、访谈、播客切片的剪辑人员。
- 希望先保留人工审核权、再用 AI 提效的团队。

## 当前版本重点

- 审核页悬浮面板改为更稳定的左右浮动按钮。
- LLM 口播标记改为保守剪辑策略：保护主线，只删高确定性冗余。
- 在线更新页支持读取服务器更新说明。
- 文档同步更新服务器发布和交接流程。
