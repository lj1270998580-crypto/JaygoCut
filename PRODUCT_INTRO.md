# Jaygo Cut 软件介绍

## 1. 软件定位
Jaygo Cut 是一款面向中文口播视频创作者的桌面端自动剪辑工具。  
目标是把“转录 -> 审核 -> 裁剪 -> 导出”做成一条可视化流水线，帮助用户更快删除静音、语气词、重复句和语义冗余内容。

## 2. 核心功能
- 本地/云端转录可切换：
  - 云端：火山引擎 ASR
  - 本地：内置 Whisper（标准 + 高精度）
- 自动预标记：
  - 静音片段（支持阈值，默认 >= 0.2 秒）
  - 语气词
  - 重复句
- LLM 智能标记：
  - 基于语义标记可删内容，支持人工二次确认
  - 提供连通测试、连通状态显示
- 审核页可视化编辑：
  - 文本与音频实时联动
  - 支持批量选择/拖拽选择
  - 颜色区分不同标记来源（静音/语气词/重复句/LLM）
- 一键执行剪辑：
  - 显示实时阶段与日志
  - 输出成片并记录历史
- 历史记录管理：
  - 打开目录
  - 恢复审核
  - 删除记录

## 3. 内置依赖（开箱即用）
已内置到install包：
- `ffmpeg`
- `ffprobe`
- Whisper 模型：
  - `ggml-base.bin`（标准）
  - `ggml-large-v3-turbo.bin`（高精度）

用户install后可直接使用本地转录与剪辑，不强依赖额外手动install。

## 5. 标准使用流程
1. 在设置里填写转录与 LLM 参数（按需）。
2. 选择视频与输出目录，点击“开始完整流程”。
3. 进入审核页，查看自动标记与 LLM 建议。
4. 人工确认后点击“执行裁剪”。
5. 在历史记录中回看、恢复或删除项目。

## 6. 连通测试说明
LLM 连通测试会校验：
- 接口地址可访问
- API Key 与模型可调用
- 响应结构是否可解析

若出现“已连通但返回空文本”，通常表示：
- 网关返回了非文本块（如仅 reasoning/thinking）
- 模型权限或路由策略限制了文本输出
- 服务端临时波动

建议优先检查：模型名、端点协议、账号权限、网关返回格式。
